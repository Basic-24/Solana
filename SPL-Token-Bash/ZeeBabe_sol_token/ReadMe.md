# Ensure you have [`solana`](https://docs.solana.com/cli/install-solana-cli-tools) and [`npm`](https://www.npmjs.com/package/download) installed in your system

## Install

```shell
npm install @solana/spl-token
```

## Install the dependencies:
```shell
npm i || npm install
```
